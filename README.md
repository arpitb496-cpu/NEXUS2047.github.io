## 🌌 Overview

**NEXUS 2047** is an Awwwards-style interactive storytelling experience that takes users on a cinematic journey through the technologies reshaping human civilization by 2047 — AI, Quantum Computing, Synthetic Biology, and beyond.

Instead of a static website, NEXUS 2047 is a **living, breathing experience** where every scroll, hover, and interaction reveals something new.

---

## ✨ Features

### 🎬 Cinematic Loading Screen
- Animated progress bar with dynamic status messages
- Smooth fade-out reveal into the main experience

### 🌟 Immersive Visuals
- **Three.js particle field** — 3,000-star background with mouse parallax
- **Live Neural Network** — animated firing signals across a multi-layer network (Ch.01)
- **Quantum Entanglement Visualizer** — rotating qubits with entanglement arcs (Ch.02)
- **DNA Double Helix** — animated nucleotide pairs with floating base-sequence readout (Ch.03)

### 🖱️ Interactive Elements
- **Custom cursor** with lagged ring and 12-dot glowing trail
- **Section indicator dots** — click to jump to any section
- **Manifesto** — words illuminate progressively as you scroll through them
- **Counter animations** — stats count up when scrolled into view
- **Hover effects** — cards lift, tiles illuminate on interaction

### 🔊 Ambient Sound Design
- Web Audio API generative drone/pulse soundscape
- Toggle button with animated sound bars indicator

### 📱 Fully Responsive
- Snap-scroll layout on desktop
- Hamburger navigation on mobile
- All canvases and layouts adapt to any screen size

### ⚡ Performance
- Single HTML file — zero build step, zero npm
- RequestAnimationFrame-driven animations
- CSS hardware acceleration throughout

## 🎯 Design Philosophy

NEXUS 2047 follows a **cyberpunk-futurist** aesthetic:
- **Palette**: Deep space black + neon cyan + magenta + amber
- **Typography**: Orbitron (display) · Bebas Neue (impact) · Space Mono (body)
- **Motion**: Every element has purpose — no animation is decorative noise
- **Depth**: Noise texture + scanlines + grid overlay create physical atmosphere
